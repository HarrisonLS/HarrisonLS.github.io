# Yarn 包管理工具

[Yarn 构建工具入门基础](https://www.jianshu.com/p/f1d96bdc545b)

### 基础操作
<br>

#### 初始化

```shell
yarn init
```

#### 添加依赖

```shell
yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]
```

#### 更新依赖

```shell
yarn upgrade [package]
yarn upgrade [package]@[version]
yarn upgrade [package]@[tag]
```

#### 删除依赖

```shell
yarn remove [package]
```

#### 安装所有依赖

```shell
yarn

// or
yarn install
```
